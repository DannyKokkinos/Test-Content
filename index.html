CTA Content

This page provides the CTA Mezzanine content hosted on this server. The content is divided into four categories, namely:

  • Original Content, where original media file without any modifications resides
  • Mezzanine Content, where various versions of the original media file with annotations and decorations reside
  • Proposed Content, where encoded streams of the mezzanine media files reside
  • Test Content, where encoded streams annotated with MPD according to CTA Content Model reside

as well as

  • Content Model, where an example stream is added with MPD according to CTA Content Model

For more information visit the GitHub page.
